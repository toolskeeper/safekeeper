# safekeeper

#### 介绍
safekeeper是一款好用的Java权限认证框架，主要解决：登录认证、权限认证、Session会话、单点登录、OAuth2.0、微服务网关鉴权 等一系列权限安全相关的框架，集成简单、开箱即用、API设计通俗易懂，通过safekeeper，你将以一种极其简单的方式实现系统的权限认证
